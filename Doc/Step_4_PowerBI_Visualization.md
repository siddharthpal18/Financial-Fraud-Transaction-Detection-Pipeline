Step_4_PowerBI_Visualization.md

Overview


A Power BI dashboard connects to the Azure SQL Database to provide visual insights into fraud patterns.  

Key Metrics & Visuals

KPIs: Tracks 98.22K Total Transactions, 4.699K Total Frauds, and 97.48K Failed Transactions.

Fraud Distribution: A pie chart shows a distribution of 95.22% Normal vs. 4.78% Fraud transactions.

Merchant Analysis: A horizontal bar chart displays the number of merchants by fraud label.

Investigation Table: Provides granular details on transaction amounts, merchant IDs, and unusual timing.
